# Evently

A simple responsive event landing page built with HTML and CSS.

## What This Project Is

A front-end practice project with:

- responsive layout
- accessible semantic structure
- mobile hamburger navigation pattern

## Features

- Responsive layout for desktop, tablet, and mobile
- Header with brand, navigation links, search field, and user icon
- Mobile hamburger menu trigger (CSS checkbox pattern)
- Event details section with inline SVG icons
- Registration form with ticket type, quantity, attendee details, and dietary needs
- “About the event” section with featured speakers and schedule highlights

## Stack

- HTML5
- CSS3 (media queries + CSS variables)
- Inline SVG

## Project Structure

```text
Evently/
├── index.html
├── styles.css
└── README.md
```

## Responsive Breakpoints

- Desktop: default/full layout
- Tablet: `<= 1024px`
- Mobile: `<= 640px`

## Notes

- This is a static front-end project (no JavaScript file)
- Form submission is set to `action="#"` for demo/prototype use.
